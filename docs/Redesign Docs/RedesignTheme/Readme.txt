This is Redesign Theme for Copernicus Land website.
We are using Bootstrap, Bootstrap Sass, Bower and Gulp.

For development, you can install the theme using:

RedesignTheme$ npm install --save-dev gulp gulp-bower gulp-sass
RedesignTheme$ bower install --save bootstrap-sass
(Install first the dependencies if missing.)

To recompile the theme after your changes run:
RedesignTheme$ gulp
